enum ResourceType

Enumération représentant les différents types de ressources utilisés dans le jeu.

Valeurs énumérées
Wood 

Bois, utilisé pour la construction ou les outils.

Stone 

Pierre, ressource de base pour les constructions solides.

Clay 

Argile, utilisée notamment pour fabriquer des briques.

Food 

Nourriture générique (peut être remplacée par des types spécifiques comme Berry, Egg...)

Berry 

Baies, ressource alimentaire récoltée dans les champs de baies.

Wool 

Laine, produite par les moutons, utilisée pour les vêtements ou le commerce.

Egg 

Œufs, ressource alimentaire produite par les poules.

Water 

Eau, ressource vitale utilisée pour les besoins des personnages.

WoodTools 

Outils en bois, utilisés dans certaines productions ou métiers.

StoneTools 

Outils en pierre, équivalents améliorés pour certaines productions.

Brick 

Briques, matériau de construction avancé fabriqué à partir d’argile.
